# DevOps Engineer Homeworks

В этом репозитории собраны домашние задания курса **«DevOps инженер»** от Нетологии.  
В ходе обучения были освоены современные практики автоматизации, управления конфигурациями, тестирования инфраструктуры и облачных технологий.

## Основные технологии и направления

- **Ansible** – написание playbook, ролей, использование vault, inventory, тестирование с Molecule и Tox.
- **Docker** – контейнеризация сервисов для локальной разработки и тестирования.
- **Yandex Cloud** – развёртывание виртуальных машин, настройка сети, безопасность, автоматизация через CLI.
- **Terraform** – управление облачной инфраструктурой (представлен в отдельных ветках).
- **ClickHouse, Vector, LightHouse** – развёртывание аналитического стека и веб‑интерфейса.
- **Git / GitHub** – контроль версий, ветвление, тегирование, pull‑request’ы.
- **CI/CD-практики** – линтинг, проверки идемпотентности, сухие прогоны (dry‑run).

Роли, созданные для Ansible, доступны в отдельных репозиториях:
